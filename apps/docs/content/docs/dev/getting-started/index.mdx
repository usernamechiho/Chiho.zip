# Initialization

Initialize the `Rehooks` configuration by running the following command into your project's directory:

```bash title="Terminal"
npx rehooks-cli@latest init
```

<Callout type="info">
  Make sure to put `@latest` in the command, as it will install the latest
  version of the CLI.
</Callout>

Learn more about the [Initialization](/docs/cli/getting-started/initialization) process.
